# 🚀 빠른 시작 가이드

## VSCode에서 바로 실행하기

### 1️⃣ 프로젝트 열기
```
VSCode → 파일 → 폴더 열기 → IMREAL 폴더 선택
```

### 2️⃣ 터미널 열기
```
단축키: Ctrl + `
```

### 3️⃣ 패키지 설치
```bash
npm install
```

### 4️⃣ 개발 서버 실행
```bash
npm run dev
```

### 5️⃣ 브라우저에서 열기
```
http://localhost:3000
```
